/**
 * @file Projektbeschreibung, Struktur und Meta-Informationen für die JSDoc Startseite.
 * @title Join – Kanban Project Management Tool
 * @project Join
 * @version 1.0.0
 * @license MIT
 * @description
 * Join ist ein leichtgewichtiges Kanban- / Task- und Kontakt-Management Tool als Multi-Page Application (MPA) mit Vanilla JavaScript (ES Modules) und Firebase Authentication.
 *
 * ## Projektübersicht
 * Nach erfolgreicher Anmeldung (oder Gastzugang) wird eine App-Shell mit Header, Sidebar und einer Summary-Ansicht gerendert. Weitere Bereiche (Board, Contacts, Add Task) sind vorbereitet und werden iterativ ergänzt.
 *
 * ## Verzeichnisstruktur (Auszug)
 *  - index.html: Einstieg (Login / App-Shell Mount)
 *  - board.html: Kanban-Board (4 Spalten)
 *  - add-task.html: Task-Erstellung Formular
 *  - contacts.html: Kontakte verwalten
 *  - script/: JavaScript-Module (main.js, auth.js, firebase.js, components/)
 *  - css/: Stylesheets (root.css, main.css, style.css)
 *  - assets/: Icons, Bilder
 *  - documentation/: Generierte JSDoc Dateien
 *
 * ## Architektur & Prinzipien
 *  - Multi-Page Application (MPA) mit separaten HTML-Dateien
 *  - Modulare ES Modules, keine Bundler-Pflicht
 *  - Firebase Authentication (E-Mail/Passwort + Gast Session)
 *  - Responsive, mobile-taugliche UI
 *  - Saubere Trennung von Darstellung (HTML/CSS) und Logik (JS)
 *
 * ## Code- & Stil-Richtlinien
 *  - Funktionen möglichst < 14 Zeilen (Single Responsibility)
 *  - Dateien überschaubar (< 400 Zeilen)
 *  - Benennung: camelCase
 *  - XSS-Vermeidung bei dynamischer Ausgabe
 *
 * ## JSDoc Konventionen
 *  - @module pro zentraler Datei (main, auth, firebase, header, sidebar, summary)
 *  - @function, @param, @returns für exportierte Funktionen
 *  - @private für interne Helfer
 *
 * ## Skripte
 *  - npm run docs        # Generiert HTML nach ./documentation
 *  - npm run docs-serve  # Generiert & startet lokalen Server
 *
 * ## Deployment
 *  - Netlify, Vercel, GitHub Pages, Firebase Hosting
 *
 * ## Zusammenarbeit
 *  - Sinnvolle Commit Messages, kleine Changes, optional Branch pro Feature
 *
 * ## Rechtliches
 *  - Datenschutzerklärung & Impressum als eigene Seiten
 *
 * Built with ❤️ using Vanilla JavaScript & Firebase
 */
